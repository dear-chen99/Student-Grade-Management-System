# 学生成绩管理系统

## 项目简介

这是一个基于 Python + ttkbootstrap 开发的桌面级学生成绩管理系统，提供完整的成绩录入、统计排名、科目分析、数据可视化、Excel 导入导出等功能。界面采用青绿色主题，操作直观，适合中小型班级或学校使用。

## 核心功能

| 功能模块           | 描述                                      |
| :------------- | :-------------------------------------- |
| **成绩录入**       | 支持行级增删改，双击单元格快速编辑，批量保存                  |
| **数据管理**       | 学生增删改查，科目管理，成绩批量设置                      |
| **统计分析**       | 总分排名、平均分排名、各科目分析（最高/最低/均分/及格率/优秀率）      |
| **数据可视化**      | 总分排名条形图、平均分对比图、科目成绩分布直方图、雷达图、箱线图        |
| **Excel 导入导出** | 生成导入模板，从 Excel 批量导入学生成绩，导出为 Excel 或 CSV |
| **班级管理**       | 查看班级列表、班级学生详情、班级平均分/最高分/最低分统计           |
| **搜索查询**       | 按学号、姓名、班级模糊搜索，显示学生详情与成绩历史               |
| **成绩预警**       | 自动识别不及格科目，在仪表盘和成绩管理页高亮显示                |
| **数据持久化**      | 自动保存至 JSON 文件，支持备份与恢复（手动备份已移除，自动保存保留）   |

## 安装与运行

### 环境要求

- **操作系统**：Windows / macOS / Linux
- **Python 版本**：3.13 或更高版本

### 安装步骤

1. **克隆或下载项目**：
   ```bash
   git clone https://gitee.com/chens-plan/student-grade-system.git
   cd student-grade-system
   ```
2. **创建并激活虚拟环境（推荐）**：
   ```bash
   python -m venv venv
   # Windows
   venv\Scripts\activate
   # macOS / Linux
   source venv/bin/activate
   ```
3. **安装依赖**：
   ```bash
   pip install -r requirements.txt
   ```
   *如果下载速度慢，可使用清华镜像源：*
   ```bash
   pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
   ```

### 运行程序

```bash
python main.py
```

*登录窗口默认用户名：`admin`，密码：`123456`。*

### 运行测试（可选）

```bash
# 运行所有测试并生成 HTML 覆盖率报告
pytest --cov=modules --cov=src --cov-report=html
```

生成的 HTML 报告位于 `htmlcov/index.html`。

## 主要功能模块说明

### 1. 仪表盘

| 卡片      | 描述            |
| :------ | :------------ |
| 👥 学生总数 | 显示系统内学生总数     |
| 📚 科目数量 | 显示已添加的科目数量    |
| 🏫 班级数量 | 显示学生归属的班级数量   |
| ⚠️ 预警人数 | 显示有不及格科目的学生人数 |

下方包含**科目平均分条形图**和**成绩预警列表**，实时反映教学概况。

### 2. 成绩录入

支持**批量录入**，每行包含行号、学号、姓名、班级、各科成绩，双击单元格可快速编辑。底部提供“加行”、“删行”、“清空”、“保存”按钮。保存时自动校验学号是否重复、成绩是否在 0-100 范围内。

### 3. Excel 导入导出

| 功能          | 描述                                          |
| :---------- | :------------------------------------------ |
| 📥 模板       | 生成包含表头（学号、姓名、班级 + 科目）的 Excel 模板             |
| 📥 导入       | 从 Excel 文件（.xlsx / .xls）导入学生数据，自动识别表头并添加新科目 |
| 📤 导出 Excel | 导出包含排名、学号、姓名、班级、各科成绩、总分、平均分、等级的 Excel 文件    |
| 📤 导出 CSV   | 导出为 CSV 格式，兼容性强                             |

### 4. 班级管理

| 信息   | 描述                      |
| :--- | :---------------------- |
| 班级名称 | 班级名称列表                  |
| 学生人数 | 班级内学生总数                 |
| 平均分  | 班级总分平均分                 |
| 最高分  | 班级最高总分                  |
| 最低分  | 班级最低总分                  |
| 操作   | 点击“查看详情”可查看班级内学生排名及成绩明细 |

### 5. 成绩管理

| 功能     | 描述                                 |
| :----- | :--------------------------------- |
| 班级过滤   | 按班级筛选学生                            |
| 成绩范围筛选 | 按不及格、及格、良好、优秀筛选                    |
| 列头排序   | 点击列头可对数据进行升序/降序排序                  |
| 双击编辑   | 双击姓名、班级或成绩单元格可直接修改                 |
| 颜色预警   | 不及格（红色）、偏低（橙色）、全优（绿色）、全部缺考（灰色）高亮显示 |

### 6. 搜索查询

支持按**学号、姓名、班级**模糊搜索，左侧显示搜索结果列表，右侧显示学生详情（学号、姓名、班级、总分、平均分）和成绩明细。点击“查看历史”可查看该生成绩修改历史。

### 7. 科目分析

| 统计项       | 描述                                         |
| :-------- | :----------------------------------------- |
| 人数        | 该科目有成绩的学生数                                 |
| 最高分 / 最低分 | 该科目的最高和最低成绩                                |
| 均分        | 平均分                                        |
| 及格率 / 优秀率 | 及格（≥60）和优秀（≥90）的百分比                        |
| 成绩分布      | 按 0-59、60-69、70-79、80-89、90-100 分段统计人数与百分比 |
| 成绩明细      | 按名次、学号、姓名、班级、成绩、等级列出所有学生                   |

支持**科目管理**（添加/删除科目）。

### 8. 图表展示

| 图表类型  | 描述                         |
| :---- | :------------------------- |
| 总分排名  | 学生总分排名条形图，前三名以金/银/铜色高亮     |
| 平均分对比 | 学生平均分对比图，标注及格线（60）和优秀线（90） |
| 成绩分布  | 指定科目的成绩分布直方图，标注平均分和及格线     |
| 雷达图   | 各科目平均分雷达图，直观展示学科强弱         |
| 箱线图   | 各科目成绩箱线图，展示分布离散程度、中位数、异常值  |
