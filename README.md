# 学生成绩管理系统

## 项目简介

这是一个基于 Python + `ttkbootstrap` + `customtkinter` 开发的**桌面级多角色成绩管理系统**。系统内置 **管理员、教师、学生** 三种独立身份，覆盖成绩录入、统计分析、Excel 导入导出、考勤管理、学生评语、通知公告、课表管理等核心功能。界面采用青绿色主题，操作直观，适合中小型学校或培训机构使用。

## 核心功能

| 功能模块           | 描述                                              |
| :------------- | :---------------------------------------------- |
| **多角色登录**      | 管理员、教师、学生三种角色独立界面，权限分明。                         |
| **成绩录入**       | 支持行级增删改，双击单元格快速编辑，批量保存，自动校验 0-100 范围。           |
| **成绩管理**       | 班级过滤、成绩范围筛选、列头排序、颜色预警（不及格红色、偏低橙色、全优绿色）。         |
| **科目分析**       | 最高/最低/均分/及格率/优秀率/成绩分布（分段统计），科目增删。               |
| **数据可视化**      | 总分排名条形图、平均分对比图、科目成绩分布直方图、雷达图、箱线图。               |
| **Excel 导入导出** | 生成导入模板，从 Excel 批量导入学生成绩，导出为 Excel 或 CSV（支持按班级）。 |
| **班级管理**       | 查看班级列表、学生详情、班级平均分/最高分/最低分，删除班级（清空学生班级信息）。       |
| **搜索查询**       | 按学号、姓名、班级模糊搜索，显示学生详情与成绩历史。                      |
| **成绩预警**       | 自动识别不及格科目，在仪表盘和成绩管理页高亮显示。                       |
| **教师专属功能**     | 授课课程管理、班级成绩统计、考勤管理（出勤/缺勤/迟到/请假）、学生评语撰写。         |
| **学生专属功能**     | 查看个人成绩、班级排名、成绩分析报告、通知公告、课表。                     |
| **管理员功能**      | 账号管理（教师/学生）、课程管理、通知发布、课表管理、系统设置（备份/恢复/重置）。      |
| **数据持久化**      | 自动保存至 JSON 文件，支持手动备份与恢复。                        |
| **头像管理**       | 支持更换管理员/教师/学生头像，自动保存至 `img/avatars/`。           |

## 安装与运行

### 环境要求

- **操作系统**：Windows / macOS / Linux
- **Python 版本**：3.13 或更高版本

### 安装步骤

1. **克隆或下载项目**：
   ```bash
   git clone https://gitee.com/chens-plan/student-grade-system.git
   cd student-grade-system
   ```
2. **创建并激活虚拟环境（推荐）**：
   ```bash
   python -m venv venv
   # Windows
   venv\Scripts\activate
   # macOS / Linux
   source venv/bin/activate
   ```
3. **安装依赖**：
   ```bash
   pip install -r requirements.txt
   ```
   *如果下载速度慢，可使用清华镜像源：*
   ```bash
   pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
   ```

### 运行程序

```bash
python main.py
```

**默认登录账号**：

- 管理员：`admin` / `123456`
- 教师：需管理员添加后使用对应工号登录（默认密码为工号）
- 学生：需管理员或教师添加后使用对应学号登录（默认密码为学号）

### 运行测试（可选）

```bash
# 运行所有测试并生成 HTML 覆盖率报告
pytest --cov=modules --cov=src --cov-report=html
```

生成的 HTML 报告位于 `htmlcov/index.html`。

## 主要功能模块使用说明

### 1. 仪表盘

- **管理员**：显示学生总数、科目数量、班级数量、预警人数、教师总数、课程总数，以及科目均分条形图和成绩预警列表。
- **教师**：显示授课课程数、教学班级数、教授学生总数、平均及格率。
- **学生**：显示已修科目数、平均分、班级排名、总分。

### 2. 成绩录入（管理员 / 教师）

支持**批量录入**，每行包含行号、学号、姓名、班级、各科成绩，双击单元格可快速编辑。底部提供“加行”、“删行”、“清空”、“保存”按钮。保存时自动校验学号是否重复、成绩是否在 0-100 范围内。

### 3. Excel 导入导出（管理员 / 教师）

| 功能          | 描述                                               |
| :---------- | :----------------------------------------------- |
| 📥 模板       | 生成包含表头（学号、姓名、班级 + 科目）的 Excel 模板                  |
| 📥 导入       | 从 Excel 文件（.xlsx / .xls）导入学生数据，自动识别表头并添加新科目      |
| 📤 导出 Excel | 导出包含排名、学号、姓名、班级、各科成绩、总分、平均分、等级的 Excel 文件，支持按班级过滤 |
| 📤 导出 CSV   | 导出为 CSV 格式，支持按班级过滤                               |

### 4. 班级管理（管理员 / 教师）

- 管理员：创建/删除班级，查看班级详情（学生排名、平均分、最高/最低分）。
- 教师：查看所授班级的统计信息。

### 5. 成绩管理（管理员 / 教师）

- **班级过滤**：按班级筛选学生。
- **成绩范围筛选**：按不及格、及格、良好、优秀筛选。
- **列头排序**：点击列头可对数据进行升序/降序排序。
- **双击编辑**：双击姓名、班级或成绩单元格可直接修改。
- **颜色预警**：不及格（红色）、偏低（橙色）、全优（绿色）、全部缺考（灰色）高亮显示。

### 6. 搜索查询（管理员 / 教师）

支持按 **学号、姓名、班级** 模糊搜索，左侧显示搜索结果列表，右侧显示学生详情（学号、姓名、班级、总分、平均分）和成绩明细。点击“查看历史”可查看该生成绩修改历史。

### 7. 科目分析（管理员）

| 统计项     | 描述                                         |
| :------ | :----------------------------------------- |
| 人数      | 该科目有成绩的学生数                                 |
| 最高分/最低分 | 该科目的最高和最低成绩                                |
| 均分      | 平均分                                        |
| 及格率/优秀率 | 及格（≥60）和优秀（≥90）的百分比                        |
| 成绩分布    | 按 0-59、60-69、70-79、80-89、90-100 分段统计人数与百分比 |
| 成绩明细    | 按名次、学号、姓名、班级、成绩、等级列出所有学生                   |

支持 **科目管理**（添加/删除科目）。

### 8. 图表展示（管理员 / 教师）

| 图表类型  | 描述                 |
| :---- | :----------------- |
| 总分排名  | 学生总分排名条形图，前三名高亮    |
| 平均分对比 | 学生平均分对比图，标注及格线和优秀线 |
| 成绩分布  | 指定科目的成绩分布直方图       |
| 雷达图   | 各科目平均分雷达图          |
| 箱线图   | 各科目成绩箱线图           |

### 9. 考勤管理（教师）

- 选择课程和日期，加载班级学生列表。
- 点击状态单元格可切换出勤/缺勤/迟到/请假。
- 支持批量设置全部出勤/全部缺勤。
- 查看考勤历史（每日统计）。

### 10. 学生评语（教师）

- 选择课程后加载班级学生列表。
- 点击学生可查看历史评语。
- 撰写新评语并保存。
- 支持编辑已有评语。

### 11. 通知公告（管理员 / 教师 / 学生）

- 管理员：发布通知（可指定接收对象：全部/仅老师/仅学生）。
- 教师/学生：查看通知列表，双击查看详情。

### 12. 课表管理（管理员 / 教师 / 学生）

- 管理员：增删改课表条目，查看变更历史。
- 教师/学生：查看课表（只读），查看变更历史。

### 13. 个人中心（管理员 / 教师 / 学生）

- 修改密码、更换头像、更新个人信息（姓名、手机、邮箱）。

### 14. 系统设置（管理员）

- **数据备份**：手动备份当前数据文件。
- **数据恢复**：从备份文件恢复数据。
- **数据重置**：清空所有数据，恢复初始状态（不可逆）。
- **查看成绩日志**：查看所有成绩修改记录。
